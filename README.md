# Hybrid Enterprise & Cloud Security Lab

## Overview

Hands-on hybrid infrastructure lab focused on Windows Server, Ubuntu, Active Directory, infrastructure automation, containerization, and cloud security.

The objective of this project is to build and document a realistic enterprise environment while developing practical experience with Windows administration, Linux administration, identity management, infrastructure as code, and platform security.
